HTML 5

HTML, este abrevierea de la HyperText Markup Language și este limbajul de marcare standard, utilizat pentru definirea documentelor, concepute pentru a fi afișate într-un browser web. De ce limbaj ”de marcare”? Fiindcă el are la bază ”marcaje” sau ”etichete” sau ”tag-uri” speciale. HTML nu este un limbaj de programare - reține aceasta!

HTML a apărut la începutul anilor ’90, ai secolului trecut – oficial în anul 1993. Inițial era utilizat pentru crearea fișierelor textuale simple, care interpretate de browser generau pagini web.
