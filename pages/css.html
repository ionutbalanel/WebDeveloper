CSS 3

CSS se descifrează ca Cascading Style Sheets, în engleză sau foi de stil în cascadă – în română, și reprezintă grupuri de proprietăţi care definesc modul de afişare al elementelor HTML. Modul de afișare se referă la: tipuri de caractere utilizate, culori, aliniere, spaţiere etc. al elementelor dintr-un document web.

În prezent, toate browserele suportă CSS, mai puțin unele versiuni mai vechi ale unor browsere.rpretate de browser generau pagini web.
