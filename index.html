Dezvoltare Web

Dezvoltarea Web, în engleză „web development”, reprezintă etapele următoare cele de
design. Web developer-ii sunt acele persoane care transformă machetele elaborate de
designer-i în site-uri web funcționale.

Astfel „web development-ul” include realizarea conținutului web, în baza machetelor
preluate de la designeri.
Dezvoltatorii web utilizează HTML, CSS JS pentru realizarea siteurilor.

Ce vom învăța?

  • Proiectarea unui site
  • Studierea limbajului HTML
  • Studierea limbajului CSS
  • Studierea limbajului JavaScript
  • Adaptarea pe dispozitive mobile
